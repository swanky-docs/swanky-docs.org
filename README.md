# swanky-docs.org

This site is built with [swanky](https://github.com/swanky-docs/swanky/). Site content is written in Markdown. Pull requests welcome!

## Development

Start the dev server at `localhost:3000`:

```
$ npm | yarn install
$ npm start
```
